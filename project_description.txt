Ανάλυση του Project: CovidApp
Το CovidApp είναι μια web εφαρμογή που σχεδιάστηκε για να βοηθά τους πολίτες και τις αρχές να παρακολουθούν περιστατικά Covid-19 και τη δημοτικότητα τοποθεσιών μέσω crowdsourcing. Παρέχει επίσης ένα admin panel για τη διαχείριση των δεδομένων.

 1. Στόχος του CovidApp
Ο βασικός στόχος της εφαρμογής είναι: ✔ Να επιτρέπει στους χρήστες να δηλώνουν περιστατικά Covid-19.
✔ Να παρέχει δεδομένα σχετικά με τη δημοτικότητα τοποθεσιών σε πραγματικό χρόνο.
✔ Να επιτρέπει στους διαχειριστές να βλέπουν στατιστικά και να διαχειρίζονται τα δεδομένα.

 Αποτελεί ένα σύστημα crowdsourcing όπου οι χρήστες ανεβάζουν δεδομένα που μπορούν να χρησιμοποιηθούν για ανάλυση.

 2. Κύριες Λειτουργίες του CovidApp
 Διαχείριση Χρηστών
✔ Οι χρήστες μπορούν να δημιουργήσουν λογαριασμό και να συνδεθούν στο σύστημα.
✔ Οι διαχειριστές (admins) έχουν επιπλέον δικαιώματα για διαχείριση των δεδομένων.

 Δηλώσεις Περιστατικών Covid-19
✔ Οι χρήστες μπορούν να υποβάλουν δήλωση αν έχουν διαγνωστεί με Covid-19.
✔ Οι δηλώσεις καταγράφονται στη βάση δεδομένων και μπορούν να χρησιμοποιηθούν για στατιστικά.
✔ Οι χρήστες μπορούν να επεξεργαστούν ή να διαγράψουν τις δηλώσεις τους.

 Παρακολούθηση Δημοτικότητας Τοποθεσιών
✔ Οι χρήστες μπορούν να υποβάλουν βαθμολογίες δημοτικότητας για διάφορες τοποθεσίες.
✔ Το σύστημα δημιουργεί ένα heatmap με βάση τη δημοτικότητα.
✔ Οι διαχειριστές μπορούν να δουν ποια μέρη είναι πιο πολυσύχναστα.

 Προβολή Στατιστικών & Ανάλυση Δεδομένων
✔ Οι διαχειριστές έχουν πρόσβαση σε στατιστικά αναφορικά με περιστατικά Covid-19.
✔ Τα δεδομένα μπορούν να απεικονιστούν με heatmaps και charts.
✔ Οι διαχειριστές μπορούν να διαχειριστούν ή να διαγράψουν δεδομένα.

 3. Αρχιτεκτονική του CovidApp
Το CovidApp αποτελείται από Frontend & Backend, με σύνδεση σε μια βάση δεδομένων MySQL.

 3.1 Frontend (Διεπαφή Χρήστη)
Γλώσσες: HTML, CSS, JavaScript.
Χρήση AJAX για δυναμική φόρτωση δεδομένων.
Heatmaps & Charts για οπτικοποίηση δεδομένων.
 3.2 Backend (Επεξεργασία & Διαχείριση Δεδομένων)
Γλώσσα: PHP.
Χρήση SQL queries για αποθήκευση και ανάκτηση δεδομένων.
API endpoints για διαχείριση δεδομένων από το frontend.
 3.3 Βάση Δεδομένων (MySQL)
✔ Πίνακες χρήστη & διαχειριστή
✔ Πίνακας δηλώσεων Covid-19
✔ Πίνακας τοποθεσιών & δημοτικότητας
✔ Πίνακας στατιστικών

 4. Ανάλυση των Κύριων Αρχείων
Το σύστημα αποτελείται από διάφορα PHP scripts που διαχειρίζονται τις λειτουργίες.

 4.1 Διαχείριση Χρηστών
 admin_login.php → Σελίδα σύνδεσης διαχειριστών.
 admin_check.php → Επαληθεύει τα διαπιστευτήρια των διαχειριστών.
 admin_logout.php → Αποσυνδέει τον διαχειριστή.

 4.2 Διαχείριση Δηλώσεων Περιστατικών
 declare.php → Φόρμα υποβολής δηλώσεων Covid-19.
 complete_declare.php → Αποθηκεύει τις δηλώσεις στη βάση.
 count_cases.php → Υπολογίζει τον αριθμό των δηλωμένων περιστατικών.

 4.3 Ανάλυση Δεδομένων & Στατιστικά
 admin_stats.php → Προβολή στατιστικών για περιστατικά Covid-19.
 mypoints.php → Εμφανίζει δεδομένα για heatmaps.
 generic.json → Περιέχει δεδομένα δημοτικότητας τοποθεσιών.

 4.4 Διαχείριση Δεδομένων από Διαχειριστές
 manage_data.php → Επιτρέπει την τροποποίηση δεδομένων.
 edit_info.php → Επεξεργασία εγγραφών.
 delete_info.php → Διαγραφή δεδομένων.
 complete_delete.php → Διαγραφή όλων των δηλώσεων από τη βάση.

 5. Οπτικοποίηση Δεδομένων
Η εφαρμογή χρησιμοποιεί heatmaps & charts για ανάλυση δεδομένων.

 

 6. Συμπέρασμα
Το CovidApp είναι ένα web σύστημα crowdsourcing που επιτρέπει: ✔ Την καταγραφή περιστατικών Covid-19.
✔ Τη συλλογή δεδομένων δημοτικότητας τοποθεσιών.
✔ Την οπτικοποίηση δεδομένων με charts & heatmaps.
✔ Τη διαχείριση στατιστικών από διαχειριστές.
